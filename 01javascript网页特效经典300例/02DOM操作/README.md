###DOM操作

文档对象模型(Document Object Model DOM)是w3c标准组织推荐使用的,用来处理可拓展标记语言(如HTML,XML)的标准编程接口,DOM以一种标准的方式来进行文档对象的处理,不以平台和语言为差异,因此各种浏览器的文档操作方式都需要遵循该标准,javascript作为浏览器中应用最为广泛的脚本语言,用它来访问和修改一个文档的内容和结构是最佳的选择,本章介绍了如何用javascript访问,创建,删除文档元素,以及如何使用javascript修改文档自身的结构
